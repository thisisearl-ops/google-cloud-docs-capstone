What's a skill

A skill is a structured bundle of guidance, steps, and resources designed to help users accomplish a specific goal—whether that's setting up a project, deploying a model, connecting services, or optimizing performance. Instead of navigating multiple pages and piecing together information, users can follow a skill to move from intent to execution in a clear, guided way.

smart_toy Empowering both Humans and AI Agents

Skills can be used by both humans and AI agents (e.g., in Vertex AI), enabling:

  • bolt Faster task completion through step-by-step, goal-oriented workflows
  • account_tree Reduced cognitive load by eliminating the need to reconstruct information across documentation
  • shield Consistent and reliable execution based on trusted, official sources
  • auto_awesome Adaptive support that works for both beginners (structured guidance) and advanced users (efficient, direct actions)
